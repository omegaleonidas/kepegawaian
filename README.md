# kepegawaian
merupakan aplikasi absensi kepegawaian berbasis android 
fitur dari aplikasi ini :
1. mengambil absensi yang berada di daerah absensi yang telah ditentukan
2. pengajuan cuti
3. riwayat absensi
4. informasi sekolah 
5. profile dari kepegawaian
